نسخه ۱ برای macOS 14+

Markdown را تمیز، سریع و فارسی‌پسند بخوان.

MD Viewer فایل‌های Markdown را با پشتیبانی از تب‌های چندگانه، Mermaid، جدول، کدهای رنگی، انتخاب متن، RTL فارسی و خروجی HTML نمایش می‌دهد.

دانلود DMG مشاهده Release
Preview Reload

خلاصه

این اپ برای خواندن سندهای فنی Markdown ساخته شده؛ مخصوصاً وقتی متن فارسی، کد، جدول و دیاگرام کنار هم هستند. 

```mermaid
flowchart TD
  A["باز کردن فایل"] --> B["Preview"]
```

امکانات

تب‌های چندگانه

چند فایل را هم‌زمان باز کن، بین آن‌ها جابه‌جا شو و هر تب را جدا reload کن.

Markdown کامل‌تر

رندر با markdown-it، جدول‌ها، task list، code block و HTML table را بهتر نمایش می‌دهد.

Mermaid و کد

دیاگرام‌های Mermaid با zoom، pan و fullscreen و کدها با highlight.js رندر می‌شوند.
